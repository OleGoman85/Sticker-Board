# 🎨 Sticker Board

A small React + TypeScript project demonstrating global state management using **Context + Provider**, persistent storage with **localStorage**, and modern UI styling with **Tailwind CSS**.

This project focuses on clean architecture, predictable state updates, and derived data patterns.

---

## 🚀 Features

- 📦 Global state management via React Context
- 💾 Persistent storage using `localStorage`
- 🎯 Sticker selection system
- ⭐ Toggle favorites
- 🔎 Search by title
- 🧠 Filter (All / Favorites)
- ➕ Add random sticker from pool
- 🗑 Delete sticker with selection fallback logic
- 🎞 Smooth hover animations
- 📐 Derived state with `useMemo`

---

Derived Values

- selectedSticker
- visibleStickers

Derived values are computed with useMemo and are never duplicated in state.

---


## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons

---
